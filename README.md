#Music App


Back-end side of music app project with Gradle dependencies manager. 


Technologies: 
- Spring Boot
- Spring Cloud
- RabbitMQ
- Eureka Discovery 
- Zuul Gateway
- Spring Security
- Spring Web
- Docker
- MySQL
- Flyway
- Lombok
- Hibernate
- Rest Repositories
